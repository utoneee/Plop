# Plop 積木拼桌
平面積木，透過拖曳擺放出不同風景。

## 功能
- **多種積木**：三角、正方、圓形、門型、拱形等……。
- **操作簡單**：積木拖曳移動，單點可旋轉45°，長按從群組拆出
- **邊對邊吸合**：積木邊對邊會像磁吸貼合，可群組拖曳或旋轉。少數形狀不支援。
- **模式-隨機盤**：開局隨機掉落 5-7 顆、無選塊介面、按補料加積木、清空＝重抽。
- **模式-自由制**：下方塊盤無限取用，上拉、點一下取塊放桌。
- **積木風格**：分成2種特性：一般（磁吸、不堆疊）／ 可堆疊：疊印、透影、顯影、消隱（積木可重疊，產生視覺效果）。
- **多種背景**：搭配不同積木風格產生視覺變化。
- **復原**：退回上一步（最多 30 步）
- **收回／刪除**：將積木拖至底部或頂部帶刪除
- **清空**：兩段式確認，防誤觸
- 拼桌狀態（含模式、風格、背景、角度）自動存在 localStorage，重開不遺失

## 使用方式
直接用瀏覽器開啟 `index.html` 即可，無需安裝、無後端。觸控優先（iPad 為主要目標裝置），滑鼠也可操作。
- 隨機盤：開局就有一桌積木，直接拼
- 點一下 → 整組轉 45°
- 拖近另一塊的邊 → 自動吸附成組
- 長按一塊 → 浮起後可單獨拖走
- 積木拖到底部或頂部 → 收回、刪除
