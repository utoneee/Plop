# Plop｜黏土拼桌

禪意的平面拼擺小品，不是解謎遊戲。紙感桌面上一盤低飽和黏土質感的幾何／有機塊，隨手拖到桌上拼排，邊對邊磁吸，無目標、無對錯、無計分。

## 功能

- **十七種黏土塊**：大三角、小三角、等腰三角、方、小方（半方塊）、長方（寬＝兩方塊、高＝半方塊）、平行四邊形、梯形、三階階梯、拱門、門、拱橋（半圓環）、半圓、圓、大圓（七巧板血統＋建築積木）＋鵝卵石、豆形（有機點綴，平滑曲線輪廓）
- **尺寸成系統**（39 半格制）：拱門 117×117＝方+小方（兩軸）、半圓與大圓直徑 117、梯形（直角梯形）底 117/頂 78/高 39 斜邊 45°、平四邊 117/高 39 斜邊 45°（橫跨 156＝兩方塊）、拱橋外弧＝半圓（翻過來合成整圓）、圓直徑 68＝門寬；門剛好塞進拱門門洞，門頂與圓靠近橋洞會自動吸成同心；梯形/平四/三角的 45° 斜邊互相咬合
- **點一下旋轉**：快速點一下塊（沒拖動）＝整組繞群組中心轉 45°
- **邊對邊磁吸**：直邊拖近直邊自動咬合對齊，塞進直角時兩條邊會同時咬上；有機形只吸不對齊（貼著挨著放）
- **群組拖動**：吸合的塊成組，拖任一塊整組一起走
- **長按拆件**：長按約半秒單塊浮起（微放大＋陰影加深），脫離群組單獨拖
- **防重疊推擠**：塊不可重疊，拖到重疊位置時像實體黏土塊一樣被推開（階梯、拱門的缺口可以塞進別的塊）
- **兩種取塊模式**：隨機盤（開局隨機掉落 10–15 顆、無選塊介面、清空＝重抽）／自由制（下方塊盤無限取用），右上角切換；小螢幕塊盤可橫向拖曳捲動（上拉取塊、點一下取塊放桌）
- **疊色開關**：無磁吸、可重疊，積木半透明、重疊處顏色相乘混色（黃＋紅＝橘）；蒙德里安的白在疊色時轉深灰、黑白灰的亮色走純半透明（搭深灰背景看）
- **三種風格**：黏土（低飽和柔影）／蒙德里安（紅黃藍白平塗＋黑粗邊）／黑白灰（無彩色灰階、無描邊）
- **四種背景**：紙／深灰／格紋／點點（風格與背景收在「外觀」按鈕裡）
- **復原**：退回上一步（最多 30 步），誤刪誤轉誤清空都救得回來
- **補料**：隨機盤模式不清空、往空位再掉 3–5 顆，拼到一半缺件時用
- **收回／刪除**：自由制拖回塊盤收回；隨機盤拖到底部刪除帶刪除
- **存圖**：把桌上構圖輸出成 PNG（含風格與背景）
- **清空**：兩段式確認，防誤觸
- 拼桌狀態（含模式、風格、背景、角度）自動存在 localStorage，重開不遺失

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，無需安裝、無後端。觸控優先（iPad 為主要目標裝置），滑鼠也可操作。

- 隨機盤：開局就有一桌積木，直接拼
- 點一下 → 整組轉 45°
- 拖近另一塊的邊 → 自動吸附成組
- 長按一塊 → 浮起後可單獨拖走
- 自由制拖回塊盤／隨機盤拖到底部 → 收回、刪除
