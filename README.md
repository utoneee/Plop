# Plop｜黏土拼桌

禪意的平面拼擺小品，不是解謎遊戲。紙感桌面上一盤低飽和黏土質感的幾何／有機塊，隨手拖到桌上拼排，邊對邊磁吸，無目標、無對錯、無計分。

## 功能

- **八種黏土塊**：大三角、小三角、方、長條、平行四邊形、半圓（七巧板血統）＋鵝卵石、豆形（有機點綴）
- **邊對邊磁吸**：直邊拖近直邊自動咬合對齊；有機形只吸不對齊（貼著挨著放）
- **群組拖動**：吸合的塊成組，拖任一塊整組一起走
- **長按拆件**：長按約半秒單塊浮起（微放大＋陰影加深），脫離群組單獨拖
- **防重疊推擠**：塊不可重疊，拖到重疊位置時像實體黏土塊一樣被推開
- **兩種取塊模式**：一盤制（固定配額，像一盒黏土）／自由制（無限取用），右上角切換
- **收回**：把塊拖回下方塊盤即收回
- **存圖**：把桌上構圖輸出成 PNG
- **清空**：兩段式確認，防誤觸
- 拼桌狀態自動存在 localStorage，重開不遺失

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，無需安裝、無後端。觸控優先（iPad 為主要目標裝置），滑鼠也可操作。

- 從下方塊盤拖一塊到桌上開始
- 拖近另一塊的邊 → 自動吸附成組
- 長按一塊 → 浮起後可單獨拖走
- 拖回塊盤 → 收回
