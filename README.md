# Plop｜黏土拼桌

平面積木小品，非解謎遊戲。
如同桌上的一盤積木，可隨手拖曳擺放，積木邊對邊能磁吸移動，有不同配色和呈現模式。

## 功能

- **十七種積木**：三角、正方、圓形、門型、拱形等……。
- **點一下旋轉**：單點一下（沒拖動）＝整組繞群組中心轉 45°
- **邊對邊磁吸**：直邊拖近直邊自動咬合對齊，塞進直角時兩條邊會同時咬上；有機形只吸不對齊（貼著挨著放）
- **群組拖動**：吸合的塊成組，拖任一塊整組一起走
- **長按拆件**：長按約半秒單塊浮起（微放大＋陰影加深），脫離群組單獨拖
- **防重疊推擠**：一般模式下機木不可重疊，拖到重疊位置時像實體被推開（階梯、拱門的缺口可以塞進別的塊）
- **兩種取塊模式**：隨機盤（開局隨機掉落 5-7 顆、無選塊介面、清空＝重抽）／自由制（下方塊盤無限取用，上拉取塊、點一下取塊放桌）
- **兩種模式**：一般（磁吸、不堆疊）／可堆疊：疊印、透影、顯影、消隱（積木可重疊，產生視覺效果）
- **九種背景**：可搭配不同積木化，模式與背景收在「外觀」按鈕裡
- **復原**：退回上一步（最多 30 步），誤刪誤轉誤清空都救得回來
- **補料**：隨機盤模式不清空、往空位再掉 3–5 顆，拼到一半缺件時用
- **收回／刪除**：將積木拖至底部或頂部帶刪除
- **清空**：兩段式確認，防誤觸
- 拼桌狀態（含模式、風格、背景、角度）自動存在 localStorage，重開不遺失

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，無需安裝、無後端。觸控優先（iPad 為主要目標裝置），滑鼠也可操作。

- 隨機盤：開局就有一桌積木，直接拼
- 點一下 → 整組轉 45°
- 拖近另一塊的邊 → 自動吸附成組
- 長按一塊 → 浮起後可單獨拖走
- 積木拖到底部或頂部 → 收回、刪除
